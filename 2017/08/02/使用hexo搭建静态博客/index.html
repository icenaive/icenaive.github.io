使用hexo搭建静态博客

2017-08-02

记录一下基本步骤，防止自己手贱！

先慢慢学习一个hexo，等技术有点进步了，去搞个wordpress玩玩！
正文：系统：Win 10
配置环境：
安装Node.js:
作用：用来生成静态页面 Node.js官网默认选项安装。
安装Git：
作用：把本地的hexo博客内容提交到GitHub上，Git官网默认选项安装。
安装淘宝npm镜像：
npm install -g cnpm –registry=https://registry.npm.taobao.org安装完以后与以前的区别为npm变为cnpm！
检查：
win+R运行->cmd->node -v npm -v git – version 成功会显示版本号！
申请Github：
第一步：注册账号。
第二步：建立仓库。
第三步：不会百度！
安装HEXO：
在电脑新建文件夹存放blog，右键git bash，
使用淘宝npm安装Hexo
命令: cnpm install -g hexo-cli
命令：cnpm install hexo –save
安装完成检查:
命令：hexo -v
本地运行：
初始化：
命令: hexo init
安装生成器:
命令：cnpm install
预览：
命令：hexo s
浏览器输入https://localhost:4000
通过Ctrl+C停止运行！
发布博客：
设置git身份信息：
命令：git config –global user.email “你的邮箱”
命令：git config –global user.name “你的用户名”
安装hexo git插件
命令：cnpm install hexo-deployer-git –save
发布更新博客：
命令： hexo d -g
写信博客：
命令：hexo new “文章名”
配置ssh：
检查ssh keys设置 检查本机密钥
命令：cd ~/.ssh
如果提示：
No such file or directory
说明第一次使用git
生成新的ssh key：
命令: ssh-keygen -t rsa -C “邮件地址”
然后回车跳过；
找到生成的id_rsa.pub文件，复制然后粘贴到GitHub保存！
